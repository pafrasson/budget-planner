## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/pafrasson/budget-planner.git
```

Entre no diretório do projeto

```bash
  cd budget-planner
```

Instale as dependências

```bash
  npm install
```

Crie um arquivo chamado *quote.db* no diretório raiz do projeto



```bash
  npm run start
```
